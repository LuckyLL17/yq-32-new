## 1. 产品概述

SciLab 是一个将复杂理科知识可视化的交互式实验平台，用户可以像玩游戏一样通过拖拽调整参数和实验道具，模拟真实实验场景，把抽象的物理/数学概念转换为直观的动态模型。

- 目标用户：中学生、大学生、理科教师、科学爱好者
- 核心价值：让抽象公式"活起来"，通过交互体验建立直觉理解，降低理科学习门槛

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览与操作所有实验 |
| 学习者 | 邮箱注册 | 保存实验参数、收藏实验 |

### 2.2 功能模块

1. **首页（实验大厅）**：实验分类导航、精选实验展示、快速入口
2. **实验工作台**：Canvas 交互画布、参数控制面板、公式实时显示、数据图表
3. **实验库**：按学科分类浏览所有可用实验

### 2.3 页面详情

| 页面 | 模块名称 | 功能描述 |
|------|----------|----------|
| 首页 | Hero 区域 | 全屏动态粒子背景 + 标题动画，展示平台理念 |
| 首页 | 实验分类卡片 | 物理/数学/化学三大分类入口，悬浮 3D 倾斜效果 |
| 首页 | 精选实验轮播 | 展示 4 个推荐实验的实时预览缩略图 |
| 实验工作台 | Canvas 画布 | 核心交互区域，支持拖拽物体、调整力/角度等参数 |
| 实验工作台 | 参数面板 | 滑块/旋钮控制，实时联动 Canvas 画面变化 |
| 实验工作台 | 公式区 | 根据当前参数实时计算并展示对应公式与数值 |
| 实验工作台 | 数据图表 | 实验数据的实时折线图/散点图 |
| 实验库 | 学科筛选 | 按物理/数学/化学分类筛选实验 |
| 实验库 | 实验卡片列表 | 每个实验的缩略图、简介、难度标签 |

## 3. 核心流程

用户进入首页 → 浏览/搜索实验 → 选择一个实验进入工作台 → 通过拖拽画布上的物体或调整参数面板 → 观察动态模型实时变化 → 查看对应公式与数据图表 → 理解抽象概念

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深邃太空黑 `#0a0e17` + 荧光青 `#00f0ff` + 能量橙 `#ff6b2b`
- **辅助色**：暗紫 `#1a1a2e`、深蓝 `#16213e`、电路绿 `#0f9b58`
- **按钮风格**：圆角微发光边框按钮，悬浮时发光增强，点击时脉冲效果
- **字体**：标题使用 Orbitron（科技感显示字体），正文使用 IBM Plex Sans
- **布局风格**：左侧导航 + 中央大画布 + 右侧参数面板，沉浸式实验空间
- **图标风格**：线性霓虹风格图标，描边发光效果

### 4.2 页面设计概览

| 页面 | 模块名称 | UI 元素 |
|------|----------|---------|
| 首页 | Hero 区域 | 全屏粒子动画背景、大字标题 Orbitron 字体、荧光描边按钮 |
| 首页 | 实验分类卡片 | 毛玻璃卡片、悬浮 3D 倾斜、学科图标发光 |
| 首页 | 精选实验轮播 | 横向滚动卡片、实时 Canvas 缩略预览、渐变边框 |
| 实验工作台 | Canvas 画布 | 深色网格背景、发光物体轨迹、拖拽手柄高亮 |
| 实验工作台 | 参数面板 | 荧光滑块、数值实时显示、重置按钮 |
| 实验工作台 | 公式区 | 半透明毛玻璃面板、LaTeX 风格公式渲染 |
| 实验工作台 | 数据图表 | 深色图表主题、荧光线条、实时数据点动画 |
| 实验库 | 学科筛选 | 标签式切换、选中态发光 |
| 实验库 | 实验卡片列表 | 网格布局、缩略图悬浮放大、难度标签色块 |

### 4.3 响应式设计

- 桌面优先设计，核心交互（Canvas 拖拽）需要足够画布空间
- 平板端参数面板折叠为底部抽屉
- 移动端画布全屏 + 浮动参数按钮

### 4.4 实验内容规划

#### 物理实验
1. **弹簧振子**：拖拽质量块、调整弹性系数，观察简谐运动
2. **抛体运动**：调整发射角度和初速度，观察弹道轨迹
3. **波的干涉**：调整双缝间距和波长，观察干涉图样
4. **万有引力**：拖拽星体，调整质量，观察轨道变化

#### 数学实验
5. **函数图像**：调整参数 a/b/c，实时观察 y=ax²+bx+c 变形
6. **分形探索**：调整迭代参数，缩放探索 Mandelbrot 集

#### 化学实验
7. **分子结构**：旋转查看 3D 分子模型，调整键角
8. **反应速率**：调整温度/浓度，观察反应速率变化

### 4.5 首期实现范围

首期聚焦 4 个核心实验，确保每个实验交互深度和视觉效果达到高品质：
1. 弹簧振子（简谐运动）
2. 抛体运动（弹道模拟）
3. 波的干涉（双缝干涉）
4. 函数图像（参数可视化）
